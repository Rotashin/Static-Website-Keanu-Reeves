2e jeu : The Matrix: Path of Neo (2005)

Fiche du jeu sur JV.com : The Matrix : Path of Neo.

2e jeu sur Matrix où il y a le personnage de Neo, le titre propose en plus quelques séquences supplémentaires qui ne figurent pas dans la saga d'origine.
Le jeu est mieux travaillé que le précédent et ça se ressent par l'accueil de ce dernier par le public.
Bon, les graphismes sont de 2005, donc faut pas s'attendre à des trucs magnifiques hein.


3e jeu : Cyberpunk 2077 (2020)

Fiche du jeu sur JV.com : Cyberpunk 2077.

Reeves joue le rôle de Johnny Silverhand (encore un John, eh beh), ce personnage apparaît directement dans le trailer du jeu à l'E3 2019.
Son apparition dans le jeu a sérieusement fait connaître le jeu sur internet alors que le studio n'était pas très connu (bon, ils ont participé au développement des jeux The Witcher, mais ils ne restaient jusque-là pas très connus).
